The Fundamentals

Before jumping into C programming, lets take a few minutes to understand its purpose, uses, and application in the real world.
C is "general-purpose", used for a wide range od applications ranging from Operating Systeam like Windows or iOS to software used in creating 3D movies.
Despite being more than 40 years old, C has maintained its popularity by being highly efficient. C progams written in one system works in another operating system without any change. This meakes Standard C Programs "portable".
Understanding C programming will do more that just help you understand how your program works, but will also be enable you to create a mental picture of how a computer works as well. The fact that the C programming language will give you a strong conceptual understanding of programming as a whole makes C one of the best langauages to start learning programming.

Is C programming right for you?

If you intend on going to a college or university to major in Electrical Engineering, Computer Engineering, or Computer Science, you will have to learn C. Typically it will be the first language you learn.
For the rest of you, C progamming is still good especially if you are still new to the world of programming.

Benefits of learning C

01 You will understand the fundamentals of how a computer works
If you have even a basic understanding of C, you will be able to visualize how a computer works including memory management and allocation. You will appreciate the flexibility that comes with the C Programming Language as opposed to its Python and Java contemporaries.
02 C is the foundation of programming
High-level programming languages like Java, Python, JavaScript etc. can interface with C progamming. You could be collaborating with another person who doesn't know anything about C, both of you will still understand and freely convey your ideas in progamming in a way you will both understand.
Moreover, a lot of popular langauges are based on C and C++ where C++ is considered a superset of C programming with Object Oriented Programming Features.
Langauges like C# and Java are cousins to C and C++ plus the syntax of JavaScript and PHP are very similar. If you know C, learning all the other langauges will come easily to you.
03 Better Understanding of Good Practices and Coding Principles
Knowing how computer works and manage memory gives you insight on how to write efficient code in other programming languages.
04 Opportunity to Impact Millions of People
In today's world, it may be easy for us to overlook C in the context that we live in a world where C programming is not even mentioned in YouTube or on Google as a way for us to develop cool projects.
Need to develop a mobile app? Learn Java for Android or Swift and Objective C for iOd. Need to build a web application? Learn PHP, ASP.net, Ruby, Python, or JavaScript and React/Angualr/Vue. So where does C programming fit in today's coding ecosystem.
For one, Python is used for making millions or a wide range of applications. C is used for making Python... Therefore, in order to contribute to Python, you need to know C programming which allows you to work on a Python interpreter that impacts millions of Python programmers. Keep in mind that this is just one of the softwares powered by C today and it impacts millions of people. Some other projects include Linux Kernel, Python Interpreter, SQLite Database & more.

Don't learn C Programming

Even if you don't know the C language, you can still make awesome software projects. Apps of today aren't built using C. They're built on JavaScript with React Native, Dart with Flutter, or natively using Swift & Java.
C takes time to master, so if you are: busy, don't want to invest time & energy, or it doesn't effect you day-to-day today to tomorrow then there is no need to learn C.
Also, if you are still really young or are a complete noob... you can start with another langauge like Python or JavaScript. Keep in mind that C is not the easiest langauge to learn.

03 Good Practices When Learning C Progamming
Looking at example code is nice, but you should run it yourself. diesmo5 has dozens of examples that will help you understand C Programming. However, if you aren't hands on while learning then you're not really practicing programming. Try to do it yourself. Visualize it, understand it, code it, modify or revise and then run it again.
Learn to use a debugging tool or a debugger which allows programmers to stop a program at any point and helps to detect and correct errors. This will make your life easier as a programmer. Knowing how to use a debugger is an essentail skill that every programming should have in their arsenal.
Understand the purpose and proper application of each method the C progamming language has to offer. Any problem can be solved by a multitude of solutions but it is an invaluable skill to be able to determine which solution is best. In simplest form, you should know when to use "while loops" and when to use "for loops".
General good practices when programming: Be consistent with the formatting, this will help in readability; Use one statement per line, this will help you write code others will understand; use proper names when naming variables, instead of using abstract variables like "x" or "a" use "counter" or "power". Your code should speak for itself and not have hundreds of lines of comments just to explain one feature.
